About Query String Comparator

Analyzing long urls, let alone comparing them, can be time consuming, difficult or even frustrating.
Ads configs, tracking urls or even cookie strings it all can come in different order so finding a particular variable value can make you stare for a while until you manage to find it.

Query String Comparator lets you compare full urls, query strings, hash values etc. by creating a readable table.

How To Use It?

  1. Copy your link into the box below.
  2. Adjust query string, variable and value separators.
  3. 'Add link to compare' or simply hit enter.
  4. Add more links
  5. Analyse and share your results.

URL & Query String

A uniform resource locator (abbreviated URL; also known as a web address, particularly when used with HTTP) is a specific character string that constitutes a reference to a resource. Most web browsers display the URL of a web page above the page in an address bar. A typical URL might look like:
http://www.example.org/wiki/Main_Page
